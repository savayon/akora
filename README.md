# Akora

논리와 존중을 기반으로 한 토론 커뮤니티 프로젝트입니다.

## 프로젝트 소개

Akora는 단순한 댓글 논쟁을 넘어, 구조화된 토론 문화를 만들기 위한 웹 커뮤니티입니다.

핵심 컨셉은 다음과 같습니다.

- 자유게시판에서 자유롭게 의견을 공유
- 필요 시 상대방에게 1:1 토론 신청
- 구조화된 토론을 통해 논리적으로 의견 교환

## 기술 스택

- Next.js
- React
- TypeScript

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버 실행 후 브라우저에서 `http://localhost:3000`으로 접속합니다.

## 프로젝트 구조

```text
app/       메인 페이지 및 라우팅
public/    정적 리소스
```

## AI 개발 규칙

AI 에이전트는 작업 전에 반드시 `AGENTS.md`의 규칙을 확인하고 이를 준수해야 합니다.
